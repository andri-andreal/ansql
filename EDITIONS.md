# ANSQL Community

ANSQL Community is a free and open-source SQL client under the [MIT License](LICENSE) — a fully functional tool for everyday relational database work.

## Features

| Area | Details |
|------|---------|
| **Database engines** | MySQL / MariaDB, PostgreSQL, SQLite |
| **Query & schema** | Query editor with autocomplete, database explorer, results grid (inline edit, add-row, range selection), designers for table / view / trigger / event / routine / sequence, basic ER diagrams |
| **Data movement** | CSV / JSON import & export |
| **Productivity** | Time Machine (journaled undo/redo), favorites, connection groups |
| **Security** | Encrypted credential vault (device mode) |
| **Localization** | Full i18n (English + Bahasa Indonesia) |
